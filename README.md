# ft_irc

**Internet Relay Chat** or **IRC** is a text-based communication protocol on the Internet.<br>
It offers real-time messaging that can be either public or private. Users can exchange direct messages and join group channels.<br><br>
IRC clients connect to IRC servers in order to join channels. IRC servers are connected together to form a network.<br><br>
**ft_irc** is an IRC server using the C++98 standard (C++98 is required :sad)
